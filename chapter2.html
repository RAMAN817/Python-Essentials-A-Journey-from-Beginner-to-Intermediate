Chapter 2: Operators and Expressions

Arithmetic Operators

Python supports the following arithmetic operators:

  • Addition: +
  • Subtraction: -
  • Multiplication: *
  • Division: /
  • Modulus (Remainder): %

Example:

a = 100
b = 20
print(a + b)   # Output: 120
print(a * b)   # Output: 2000
print(a - b)   # Output: 80
print(a % b)   # Output: 0  # Remainder

Division and Floor Division Operators

The division operator / always returns a floating-point value. The floor division operator // returns the largest integer less than or equal to the division result.

a = 100
b = 20
print(a / b)       # Output: 5.0 (Division always returns a float)
print(10 // 2)     # Output: 5   (Floor division returns an int if both operands are int)
print(10.0 // 2)   # Output: 5.0 (Floor division returns a float if one operand is float)
print(10 / 3)      # Output: 3.3333333333333335
print(10.0 / 3)    # Output: 3.3333333333333335
print(10 // 3)     # Output: 3
print(10.0 // 3)   # Output: 3.0

Concatenating strings and numbers requires explicit conversion:

print('roman' + str(10))  # Output: 'roman10'

Relational Operators

Relational operators compare values and return a Boolean result (True or False):

  • ==: Equal to
  • !=: Not equal to
  • <: Less than
  • >: Greater than
  • <=: Less than or equal to
  • >=: Greater than or equal to

Examples with integers:

a = 10
b = 20
print(a > b)   # Output: False
print(a < b)   # Output: True
print(a <= b)  # Output: True
print(a >= b)  # Output: False

Examples with strings (based on Unicode values):

s1 = 'roman'
s2 = 'durga'
print(s1 > s2)  # Output: True

s3 = 'Roman'
s4 = 'roman'
print(s3 > s4)  # Output: False

Examples with Booleans:

print(True > False)  # Output: True

Chained comparisons:

print(10 < 20 < 30)  # Output: True
print(10 < 20 < 30 > 40)  # Output: False

Equality Operators

Equality operators check if values are equal or not:

  • ==: Equal to
  • !=: Not equal to

Examples:

print(10 == 20)       # Output: False
print(10 != 20)       # Output: True
print(10 == 10.0)     # Output: True
print(1 == True)      # Output: True

Difference Between is and ==

- is: Checks if two variables point to the same object (identity comparison).

- ==: Checks if the values of two objects are equal (value comparison).

Example:

l1 = [10, 20, 30, 40]
l2 = [10, 20, 30, 40]
print(l1 is l2)   # Output: False (different objects)
print(l1 == l2)   # Output: True  (same content)

l3 = l1
print(l1 is l3)   # Output: True  (same object)

Logical Operators

Logical operators are used to combine conditional statements:

  • and: Returns True if both operands are True
  • or: Returns True if at least one operand is True
  • not: Returns the opposite Boolean value

Examples with Booleans:

print(not True)       # Output: False
print(True and False)  # Output: False
print(True or False)   # Output: True

User Authentication Example:

username = input('Enter Username: ')
password = input('Enter Password: ')

if username == 'durga' and password == 'sunny':
    print('Valid user')
else:
    print('Invalid user')

Logical Operators with Non-Boolean Types:

In Python, logical operators can be used with non-boolean types, where:

  • Zero, empty strings, and empty collections are considered False.
  • Non-zero numbers and non-empty objects are considered True.

and Operator:

The result is the first operand if it is False; otherwise, it is the second operand.

print(0 and 10)            # Output: 0
print(20 and 10)           # Output: 10
print('roman' and 'son')   # Output: 'son'
print('' and 'roman')      # Output: ''

or Operator:

The result is the first operand if it is True; otherwise, it is the second operand.

print(10 or 20)   # Output: 10
print(0 or 20)    # Output: 20

not Operator:

The result is always a Boolean value.

print(not 'roman')  # Output: False
print(not '')       # Output: True
print(not 0)        # Output: True
print(not 10)       # Output: False

Bitwise Operators

Bitwise operators perform operations on binary representations of integers and Booleans:

  • &: Bitwise AND
  • |: Bitwise OR
  • ^: Bitwise XOR
  • ~: Bitwise NOT (complement)
  • <<: Left shift
  • >>: Right shift

Examples:

# Bitwise AND
print(5 & 4)  # Output: 4

# Calculation:
# 5 in binary: 101
# 4 in binary: 100
# Result:      100 (which is 4)

Bitwise Complement (~)

The bitwise complement of an integer n is equal to -(n+1).

print(~4)  # Output: -5
# Explanation:
# ~4 = -(4 + 1) = -5

Shift Operators:

Left Shift (<<): Shifts bits to the left, filling vacant bits with zeros.

print(10 << 2)  # Output: 40
# 10 in binary: 0000...1010
# After shifting: 0010...1000 (which is 40)

Right Shift (>>): Shifts bits to the right, filling vacant bits with zeros.

print(10 >> 2)  # Output: 2
# 10 in binary: 0000...1010
# After shifting: 0000...0010 (which is 2)

Bitwise Operators with Booleans:

  • True & False: False
  • True | False: True
  • True ^ False: True
  • ~True: -2 (since ~True equals -(1 + 1))

Assignment Operators

Assignment operators are used to assign values to variables:

  • =: Simple assignment
  • Compound assignment operators combine an operation with assignment:

Examples of compound assignment operators:

  • +=, -=, *=, /=, %=, //=, **=, &=, |=, ^=, >>=, <<=

Examples:

x = 10
x += 20
print(x)  # Output: 30

x = 10
x &= 5
print(x)  # Output: 0

Note: Python does not support increment (x++) or decrement (x--) operators.

Ternary Operator

A ternary operator in Python allows for conditional assignment or return of a value. The syntax is:

variable = value_if_true if condition else value_if_false

Example:

a, b = 10, 20
x = 30 if a < b else 40
print(x)  # Output: 30

Nesting Ternary Operators:

a = int(input("Enter First Number: "))
b = int(input("Enter Second Number: "))
c = int(input("Enter Third Number: "))

minimum = a if a < b and a < c else b if b < c else c
print("Minimum Value:", minimum)

Another Example:

a = int(input("Enter First Number: "))
b = int(input("Enter Second Number: "))

print("Both numbers are equal" if a == b else "First Number is less than Second Number" if a < b else "First Number is greater than Second Number")

Special Operators

Identity Operators

Identity operators compare the memory addresses of two objects:

  • is: Returns True if both variables point to the same object.
  • is not: Returns True if they do not point to the same object.

Examples:

a = 10
b = 10
print(a is b)  # Output: True

x = True
y = True
print(x is y)  # Output: True

a = "durga"
b = "durga"
print(a is b)  # Output: True

list1 = ["one", "two", "three"]
list2 = ["one", "two", "three"]
print(list1 is list2)     # Output: False (different objects)
print(list1 == list2)     # Output: True  (same content)

Membership Operators

Membership operators check if a value is a member of a sequence (string, list, tuple, set, dictionary):

  • in: Returns True if the value is present.
  • not in: Returns True if the value is not present.

Examples:

x = "Hello, learning Python is very easy!"
print('h' in x)          # Output: True
print('d' in x)          # Output: False
print('d' not in x)      # Output: True
print('Python' in x)     # Output: True

list1 = ["sunny", "bunny", "chinny", "pinny"]
print("sunny" in list1)   # Output: True
print("tunny" in list1)   # Output: False
print("tunny" not in list1)  # Output: True

Operator Precedence

Operator precedence determines the order in which operations are performed. Parentheses () have the highest precedence, followed by exponentiation **, and so on.

Refer to the Python documentation for the complete operator precedence table.

Mathematical Functions (math Module)

The math module provides access to mathematical functions:

Importing the math Module:

import math

Common Functions:

  • ceil(x): Returns the smallest integer greater than or equal to x.
  • floor(x): Returns the largest integer less than or equal to x.
  • pow(x, y): Returns x raised to the power y.
  • factorial(x): Returns the factorial of x.
  • trunc(x): Truncates the decimal part of x.
  • gcd(x, y): Returns the greatest common divisor of x and y.
  • sin(x), cos(x), tan(x): Trigonometric functions.

Constants:

  • pi: 3.141592653589793
  • e: 2.718281828459045
  • inf: Infinity
  • nan: Not a number

Example: Calculating the Area of a Circle

from math import pi

r = 16
area = pi * r ** 2
print("Area of the circle is:", area)
# Output: Area of the circle is: 804.247719318987

Input and Output Statements

Input Functions

In Python 3.x, input() reads input from the user and returns it as a string.

name = input("Enter your name: ")
print("Hello,", name)

Reading Multiple Values

Using list comprehension and split():

a, b = [int(x) for x in input("Enter two numbers: ").split()]
print("Product is:", a * b)

Using eval() Function

The eval() function evaluates the input string as a Python expression.

x = eval(input("Enter something: "))
print(type(x))

Command Line Arguments

Command line arguments are values passed to a script when it is executed. They are available via the sys.argv list.

Example:

import sys

print("Number of arguments:", len(sys.argv))
print("Argument List:", sys.argv)

Summing Command Line Arguments:

import sys

args = sys.argv[1:]
total = 0
for x in args:
    total += int(x)
print("The sum:", total)

Output Statements

The print() function outputs data to the console.

Using sep Parameter:

print('Python', 'is', 'fun', sep=' - ')
# Output: Python - is - fun

Using end Parameter:

print("Hello", end=' ')
print("World")
# Output: Hello World

Formatted Strings (Old Style)

Using format specifiers:

  • %i or %d: Integer
  • %f: Float
  • %s: String
a = 10
b = 20
c = 30
print("a value is %d" % a)
print("b value is %d and c value is %d" % (b, c))
# Output:
# a value is 10
# b value is 20 and c value is 30

Formatted Strings (New Style)

Using the format() method:

name = "Durga"
salary = 10000
gf = "Sunny"

print("Hello {0}, your salary is {1} and your friend {2} is waiting.".format(name, salary, gf))
print("Hello {x}, your salary is {y} and your friend {z} is waiting.".format(x=name, y=salary, z=gf))
# Output:
# Hello Durga, your salary is 10000 and your friend Sunny is waiting.
# Hello Durga, your salary is 10000 and your friend Sunny is waiting.
