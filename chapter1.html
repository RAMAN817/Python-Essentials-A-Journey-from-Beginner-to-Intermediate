Data Types Python

Introduction

Python is a high-level, general-purpose programming language known for its simplicity and readability. It is widely used for developing applications due to its easy-to-understand syntax and dynamic typing. Low-level activities like memory management and object destruction are handled by the language itself, making it programmer-friendly. Examples of high-level languages include Python, Java, and C#.

Why Named Python?

Guido van Rossum, the creator of Python, named the language after the British comedy show "Monty Python's Flying Circus." He was a fan of the show and wanted a name that was short, unique, and slightly mysterious.

Python as an All-Rounder

  • Functional programming features from languages like Haskell.
  • Object-oriented programming features from languages like C++.
  • Scripting language features from languages like Perl and shell scripts.
  • Modular programming features from Modula-3.

Python Features

  1. Simple and Easy to Learn: Python has a minimalistic syntax with only 33 reserved words, making it easier to learn and write code.
  2. Free and Open Source: Python is maintained by the Python Software Foundation, a non-profit organization. It is freely available for everyone to use and modify.
  3. High-Level Language: Python abstracts away low-level details like memory management, allowing programmers to focus on problem-solving.
  4. Platform Independence: Python code can run on various platforms without modification due to the presence of the Python Virtual Machine (PVM).
  5. Portability: Python programs can be easily moved from one platform to another without any changes.
  6. Dynamically Typed: Variables in Python do not need explicit declarations; the type is determined at runtime.
  7. Supports Multiple Programming Paradigms: Python supports procedural, object-oriented, and functional programming styles.
  8. Interpreted Language: Python code is executed line by line, which simplifies debugging.
  9. Extensible: Python can incorporate code from other languages like C or C++ to optimize performance.
  10. Embedded: Python code can be embedded within applications written in other languages to provide scripting capabilities.
  11. Extensive Library Support: Python has a rich set of libraries for various domains like web development, data analysis, machine learning, and more.

Limitations of Python

  1. Mobile Application Development: Python has limited library support for mobile applications compared to languages like Swift or Kotlin.
  2. Enterprise-Level Applications: For large-scale applications requiring extensive code, Python might not be as efficient due to its interpreted nature.
  3. Performance: Being an interpreted language, Python is generally slower in execution compared to compiled languages like C or C++.

Flavors of Python

Python's open-source nature has led to various customized versions:

  • CPython: The standard implementation written in C.
  • Jython: Python for Java applications.
  • IronPython: Python for .NET framework applications.
  • RubyPython: Integrates Python and Ruby languages.
  • Anaconda Python: A distribution for large-scale data processing and scientific computing.
  • Stackless Python: For concurrency and parallelism.
  • PyPy: A fast Python implementation with a Just-In-Time compiler.

Major Python Versions

  • Python 0.9.0: Released on February 20, 1991.
  • Python 1.0: Released in January 1994.
  • Python 2.0: Released on October 16, 2000.
  • Python 3.0: Released on December 3, 2008.

Note that Python 3.x is not backward compatible with Python 2.x. This allows Python 3.x to introduce new features and improvements without being constrained by legacy code.

Python IDLE

IDLE stands for Integrated Development and Learning Environment. It is a simple IDE that comes bundled with Python, allowing you to write, execute, and debug Python code in an interactive environment.

IDLE is a REPL (Read-Eval-Print Loop) tool, suitable for learning and experimenting with small code snippets. For larger applications, consider using full-featured IDEs like PyCharm, Eclipse with PyDev, or VSCode.

Python Identifiers

Identifiers are names used to identify variables, functions, classes, modules, and other objects in Python. Here are the rules for creating identifiers:

  1. Can contain letters (a-z, A-Z), digits (0-9), and underscores (_).
  2. Cannot start with a digit.
  3. Case-sensitive (e.g., variable, Variable, and VARIABLE are different).
  4. Cannot use Python reserved words as identifiers.

Python Reserved Words

Reserved words (keywords) have special meaning in Python and cannot be used as identifiers. Some of the Python keywords include:

  • False, True, None
  • and, or, not, in, is
  • if, elif, else
  • for, while, break, continue
  • def, class, with, pass, lambda
  • return, yield
  • import, from, as

Python Data Types

Python supports various data types, and you do not need to declare the type explicitly. The interpreter infers the type at runtime, making Python a dynamically typed language.

Numeric Types

  • Integer (int): Whole numbers, e.g., 123, -456.
  • Floating-point (float): Numbers with decimal points, e.g., 3.14, -0.001.
  • Complex (complex): Numbers with real and imaginary parts, e.g., 4+5j.

Boolean Type

  • Boolean (bool): Represents truth values, either True or False.

String Type

  • String (str): A sequence of characters enclosed in single, double, or triple quotes.

Sequence Types

  • List (list): Ordered, mutable collections, e.g., [1, 2, 3].
  • Tuple (tuple): Ordered, immutable collections, e.g., (1, 2, 3).
  • Range (range): Represents an immutable sequence of numbers, commonly used in loops.

Set Types

  • Set (set): Unordered collections of unique elements, e.g., {1, 2, 3}.
  • Frozen Set (frozenset): Immutable version of a set.

Mapping Type

  • Dictionary (dict): Unordered collections of key-value pairs, e.g., {'a': 1, 'b': 2}.

Type Casting

Type casting refers to converting one data type to another. Common functions include:

  • int(): Converts to an integer.
  • float(): Converts to a floating-point number.
  • complex(): Converts to a complex number.
  • str(): Converts to a string.
  • bool(): Converts to a boolean.

Collection Related Data Types

List

Lists are ordered, mutable collections that allow duplicate elements. They are defined using square brackets: 

my_list = [1, 2, 3, 4]

Tuple

Tuples are ordered, immutable collections that allow duplicate elements. They are defined using parentheses: 

my_tuple = (1, 2, 3, 4)

Set

Sets are unordered collections of unique elements. They are defined using curly braces: 

my_set = {1, 2, 3, 4}

Dictionary

Dictionaries store key-value pairs and are defined using curly braces: 

my_dict = {'name': 'Alice', 'age': 30}

Escape Characters

Common escape characters in Python strings include:

  • \n: New line
  • \t: Tab
  • \\: Backslash
  • \': Single quote
  • \": Double quote

Comments

Comments are used to explain code and are ignored by the interpreter.

  • Single-line comments: Start with the hash symbol (#).
  • Multi-line comments: Use triple quotes (''' or """). However, in Python, these are actually multi-line strings and are not true comments.

Constants in Python

Python does not have built-in constant types, but by convention, constants are declared and assigned in capital letters: 

PI = 3.14159
