Chapter 4: More on Strings

Introduction

In Python, strings are one of the most commonly used data types. Approximately 90% of objects in any programming language are string objects. In Python, there is no separate character data type; a single character is simply a string of length one.

A string in Python is a sequence of characters enclosed within single quotes ' or double quotes ". Multi-line strings can be created using triple quotes:

multi_line_str = '''Roman
Son
Thapa'''

Accessing Characters in a String

1. By Using Index

Python strings support both positive and negative indexing:

s = 'durga'
# Indexing:
#  D    U    R    G    A
#  0    1    2    3    4   (Positive Index)
# -5   -4   -3   -2   -1   (Negative Index)

print(s[0])   # Output: 'd'
print(s[-1])  # Output: 'a'
print(s[100]) # Raises IndexError

Example: Display Characters with Positive and Negative Indices

s = input('Enter a string: ')
i = 0
for x in s:
    print('The character at +ve index {} and -ve index {} is: {}'.format(i, i - len(s), x))
    i += 1

2. By Using Slice Operator

Syntax:

s[begin:end:step]
  • begin: Starting index (default is 0 if omitted)
  • end: Ending index (default is len(s) if omitted)
  • step: Step size (default is 1 if omitted)

The slice will include characters from index begin to end - 1.

Examples:

s = 'abcdefghijk'
print(s[2:7])    # Output: 'cdefg'
print(s[:7])     # Output: 'abcdefg'
print(s[2:])     # Output: 'cdefghijk'
print(s[:])      # Output: 'abcdefghijk'
print(s[2:7:2])  # Output: 'ceg'
print(s[::2])    # Output: 'acegik'

Slice Operator Rules

  • begin, end, and step can be positive or negative.
  • If step is zero, it raises a ValueError.
  • Default values:
    • Forward direction (positive step): begin=0, end=len(s)
    • Backward direction (negative step): begin=-1, end=-(len(s)+1)

String Operators

Concatenation (+)

Combines two strings into one.

print('Hello' + ' World')  # Output: 'Hello World'

Repetition (*)

Repeats a string multiple times.

print('Hello' * 3)  # Output: 'HelloHelloHello'

Membership Operators

  • in: Returns True if a substring is found in the string.
  • not in: Returns True if a substring is not found in the string.
print('a' in 'durga')  # Output: True

s = input("Enter the main string: ")
subs = input("Enter the substring to search: ")
if subs in s:
    print('Substring is present in the string')
else:
    print('Substring is not present in the string')

String Comparison

We can use comparison operators (<, <=, >, >=) and equality operators (==, !=) for strings. Comparison is based on lexicographical order (Unicode code points).

Example:

s1 = input("Enter first string: ")
s2 = input("Enter second string: ")

if s1 == s2:
    print("Both strings are equal")
elif s1 < s2:
    print("First string is less than second string")
else:
    print("First string is greater than second string")

Removing Spaces from a String

Methods to remove spaces:

  • lstrip(): Removes leading spaces.
  • rstrip(): Removes trailing spaces.
  • strip(): Removes both leading and trailing spaces.
city = input('Enter the name of the city: ').strip()
if city == 'Banepa':
    print('Gokul Baskota')
elif city == 'Bhojpur':
    print('Sudan Kirati')
else:
    print('KP Oli')

Finding Substrings

1. find() Method

Searches for a substring from left to right and returns the index of the first occurrence. Returns -1 if not found.

s = 'ABCBA'
print(s.find('B'))   # Output: 1
print(s.find('Z'))   # Output: -1

2. rfind() Method

Searches for a substring from right to left and returns the index of the first occurrence from the right. Returns -1 if not found.

print(s.rfind('B'))  # Output: 3

3. index() Method

Similar to find() but raises a ValueError if the substring is not found.

print(s.index('B'))   # Output: 1
# print(s.index('Z'))  # Raises ValueError

4. rindex() Method

Similar to rfind() but raises a ValueError if the substring is not found.

print(s.rindex('B'))  # Output: 3

Example: Email Validation Using index() Method

email = input('Enter your email ID: ')
try:
    i = email.index('@')
    print('Valid email ID')
except ValueError:
    print('Invalid email ID')

Counting Substrings

Use the count() method to count occurrences of a substring:

s = 'ABCDABCD'
print(s.count('A'))     # Output: 2
print(s.count('Z'))     # Output: 0

Example: Find All Occurrences of a Substring

s = 'ABCABCABCA'
subs = input('Enter substring to search: ')
i = s.find(subs)
if i == -1:
    print('Specified substring not found')
else:
    while i != -1:
        print('{} found at index: {}'.format(subs, i))
        i = s.find(subs, i + len(subs), len(s))
    print('Total occurrences:', s.count(subs))

Replacing Substrings

The replace() method replaces occurrences of a substring with another substring:

s = 'ABABABA'
s_new = s.replace('A', 'B')
print(s_new)  # Output: 'BBBBBBB'

Note: Strings are immutable in Python. Methods like replace() create a new string.

s = 'ABABABAB'
print('Before replacement:', id(s))
s = s.replace('A', 'B')
print('After replacement:', id(s))

Splitting and Joining Strings

Splitting Strings

The split() method divides a string into a list of substrings based on a separator (default is space).

s = 'Durga Software Solutions'
lst = s.split()
print(lst)  # Output: ['Durga', 'Software', 'Solutions']

Joining Strings

The join() method combines a list of strings into a single string with a specified separator.

lst = ['Roman', 'Son', 'Thapa']
s = ''.join(lst)
print(s)  # Output: 'RomanSonThapa'

Changing Case of Characters

  • upper(): Converts all characters to uppercase.
  • lower(): Converts all characters to lowercase.
  • swapcase(): Swaps case of all characters.
  • title(): Capitalizes the first character of each word.
  • capitalize(): Capitalizes the first character of the string.
s = 'Learning Python is very easy'
print(s.upper())       # Output: 'LEARNING PYTHON IS VERY EASY'
print(s.lower())       # Output: 'learning python is very easy'
print(s.swapcase())    # Output: 'lEARNING pYTHON IS VERY EASY'
print(s.title())       # Output: 'Learning Python Is Very Easy'
print(s.capitalize())  # Output: 'Learning python is very easy'

Example: Case-Insensitive String Comparison

s1 = input('Enter first string: ').lower()
s2 = input('Enter second string: ').lower()
if s1 == s2:
    print('Yes, both are equal')
else:
    print('No, the strings are different')

Example: Validate Username and Password

Username is case-insensitive, but password is case-sensitive.

username = input('Enter username: ').lower()
password = input('Enter password: ')
if username == 'durga' and password == 'anushka':
    print('Valid user')
else:
    print('Invalid user')

Example: Convert First and Last Character to Uppercase

s = input('Enter a string: ')
output = s[0].upper() + s[1:len(s)-1] + s[-1].upper()
print('Output:', output)

Checking Starting and Ending Substrings

  • startswith(substring): Checks if the string starts with the specified substring.
  • endswith(substring): Checks if the string ends with the specified substring.
s = 'Learning with Durgasoft is fun'
print(s.startswith('Learning'))  # Output: True
print(s.endswith('fun'))         # Output: True
print(s.endswith('boring'))      # Output: False

Checking the Type of Characters in a String

  • isalnum(): Returns True if all characters are alphanumeric.
  • isalpha(): Returns True if all characters are alphabetic.
  • islower(): Returns True if all characters are lowercase.
  • isupper(): Returns True if all characters are uppercase.
  • isdigit(): Returns True if all characters are digits.
  • istitle(): Returns True if the string is in title case.
  • isspace(): Returns True if the string contains only whitespace.

Examples:

print('Durga786'.isalnum())    # Output: True
print('durga786'.isalpha())    # Output: False
print('durga'.isalpha())       # Output: True
print('durga'.isdigit())       # Output: False
print('786786'.isdigit())      # Output: True
print('abc'.islower())         # Output: True
print('Abc'.islower())         # Output: False
print('abc123'.islower())      # Output: True
print('ABC'.isupper())         # Output: True
print('Learning python is Easy'.istitle())  # Output: False
print('Learning Python Is Easy'.istitle())  # Output: True
print('   '.isspace())         # Output: True

Demo Program:

char = input("Enter any character: ")
if char.isalnum():
    print("Alphanumeric character")
    if char.isalpha():
        print("Alphabet character")
        if char.islower():
            print("Lowercase alphabet character")
        else:
            print("Uppercase alphabet character")
    else:
        print("Digit character")
elif char.isspace():
    print("It is a space character")
else:
    print("Non-space special character")
