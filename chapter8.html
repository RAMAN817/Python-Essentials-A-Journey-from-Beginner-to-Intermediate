Chapter 8: Dictionaries

Introduction to Dictionaries

A dictionary in Python is a collection of key-value pairs. It's an unordered, mutable, and indexed collection. Dictionaries are also known in other programming languages as:

  • C++/Java: Map
  • Perl/Ruby: Hash

Examples of real-world dictionaries include:

  • Phone directory: mapping mobile numbers to names.
  • DNS records: mapping IP addresses to domain names.

Characteristics of Dictionaries

  1. Key-Value Pairs: Each element is a pair consisting of a key and a value.
  2. Duplicate Keys Not Allowed: Keys must be unique within a dictionary, but values can be duplicated.
  3. Insertion Order Not Preserved: Prior to Python 3.7, dictionaries did not preserve insertion order. From Python 3.7 onwards, insertion order is preserved as an implementation detail. However, it's best not to rely on this behavior in code meant to be portable across Python versions.
  4. No Indexing or Slicing: Dictionaries are unordered (up to Python 3.6), so indexing and slicing operations are not applicable.
  5. Heterogeneous Objects: Both keys and values can be of any data type.
  6. Mutable and Dynamic: Dictionaries can grow and shrink as needed. They are mutable, so you can change their content after creation.
  7. Dictionary Syntax: Defined using curly braces {} with key-value pairs separated by colons :.

Creating Dictionary Objects

1. Empty Dictionary

d = {}
# or
d = dict()

2. With Known Data

d = {100: 'Alice', 200: 'Bob', 300: 'Charlie'}

3. Using the dict() Function

You can create a dictionary from a sequence of key-value pairs.

l = [(100, 'A'), (200, 'B'), (300, 'C')]  # List of tuples
d = dict(l)
print(d)  # Output: {100: 'A', 200: 'B', 300: 'C'}

Note: The sequence should contain elements with exactly two values (key and value). Possible combinations include:

  • List of tuples
  • Tuple of tuples
  • List of lists
  • Tuple of lists
  • Set of tuples

4. Using Dynamic Input

d = eval(input('Enter the dictionary: '))

Accessing Data from a Dictionary

To retrieve the value associated with a key:

d = {100: 'Alice', 200: 'Bob', 300: 'Charlie'}
print(d[100])  # Output: 'Alice'
# print(d[400])  # Raises KeyError if the key is not present

To avoid a KeyError, you can check if the key exists using the in operator:

key = int(input('Enter the key to find the value: '))
if key in d:
    print('The corresponding value of the key is', d[key])
else:
    print('The specified key is not available')

Adding or Updating Data in a Dictionary

To add a new key-value pair or update an existing key:

d[key] = value

Example:

d = {100: 'Alice', 200: 'Bob'}
d[300] = 'Charlie'  # Adding a new key-value pair
print(d)  # Output: {100: 'Alice', 200: 'Bob', 300: 'Charlie'}

d[200] = 'Bobby'    # Updating an existing key
print(d)  # Output: {100: 'Alice', 200: 'Bobby', 300: 'Charlie'}

Deleting Data from a Dictionary

To delete a key-value pair:

del d[key]

Example:

d = {100: 'Alice', 200: 'Bob', 300: 'Charlie'}
del d[200]
print(d)  # Output: {100: 'Alice', 300: 'Charlie'}

Deleting multiple keys:

del d[100], d[300]
print(d)  # Output: {}

Note: Attempting to delete a key that doesn't exist will raise a KeyError.

Example: Storing Student Names and Marks

n = int(input('Enter number of students: '))
d = {}
for i in range(n):
    name = input('Enter the name of the student: ')
    marks = int(input('Enter marks of the student: '))
    d[name] = marks

print('*' * 30)
print('NAME', '\t\t', 'MARKS')
print('*' * 30)
for name in d:
    print(name, '\t\t', d[name])

Operators with Dictionaries

Mathematical Operators

Operators like + and * are not applicable to dictionaries and will raise a TypeError if used.

Equality Operator

You can use == and != to compare two dictionaries. They are equal if they have the same key-value pairs, regardless of the order.

d1 = {100: 'Alice', 200: 'Bob'}
d2 = {200: 'Bob', 100: 'Alice'}
print(d1 == d2)  # Output: True

Relational Operators

Operators like <, >, <=, and >= are not applicable to dictionaries.

Membership Operator

The in and not in operators check for the presence of keys in the dictionary.

d = {100: 'Alice', 200: 'Bob'}
print(100 in d)      # Output: True
print(300 not in d)  # Output: True

Important Methods and Functions

1. len() Function

Returns the number of key-value pairs in the dictionary.

d = {100: 'Alice', 200: 'Bob', 300: 'Charlie'}
print(len(d))  # Output: 3

2. get() Method

Retrieves the value associated with a key. Returns None (or a default value) if the key is not found, avoiding a KeyError.

d = {100: 'Alice', 200: 'Bob'}
print(d.get(100))               # Output: 'Alice'
print(d.get(300))               # Output: None
print(d.get(300, 'Not Found'))  # Output: 'Not Found'

3. update() Method

Updates the dictionary with key-value pairs from another dictionary.

d1 = {100: 'Alice', 200: 'Bob'}
d2 = {200: 'Bobby', 300: 'Charlie'}
d1.update(d2)
print(d1)  # Output: {100: 'Alice', 200: 'Bobby', 300: 'Charlie'}

Dictionary Views

1. keys() Method

Returns a view object containing the dictionary's keys.

d = {100: 'Alice', 200: 'Bob', 300: 'Charlie'}
keys = d.keys()
print(keys)  # Output: dict_keys([100, 200, 300])

# Iterating over keys
for key in d.keys():
    print(key)

2. values() Method

Returns a view object containing the dictionary's values.

values = d.values()
print(values)  # Output: dict_values(['Alice', 'Bob', 'Charlie'])

# Iterating over values
for value in d.values():
    print(value)

3. items() Method

Returns a view object containing the dictionary's key-value pairs as tuples.

items = d.items()
print(items)  # Output: dict_items([(100, 'Alice'), (200, 'Bob'), (300, 'Charlie')])

# Iterating over items
for key, value in d.items():
    print(key, '->', value)

Example: Retrieving Value for a Given Key

d = {100: 'Alice', 200: 'Bob', 300: 'Charlie'}
key = int(input('Enter key to get the value: '))
if key in d:
    print('The corresponding value:', d.get(key))
else:
    print('The specified key is not present')

Example: Retrieving Key for a Given Value

d = {100: 'Alice', 200: 'Bob', 300: 'Charlie'}
value = input('Enter value to find the key: ')
found = False
for k, v in d.items():
    if v == value:
        print('The corresponding key:', k)
        found = True
        break
if not found:
    print('The specified value is not found in the dictionary')

Removing Items from a Dictionary

1. pop() Method

Removes the item with the specified key and returns its value. Raises a KeyError if the key is not found.

d = {100: 'Alice', 200: 'Bob', 300: 'Charlie'}
value = d.pop(300)
print(value)  # Output: 'Charlie'
print(d)      # Output: {100: 'Alice', 200: 'Bob'}

# Using default value to avoid KeyError
value = d.pop(400, 'Not Found')
print(value)  # Output: 'Not Found'

2. popitem() Method

Removes and returns an arbitrary key-value pair as a tuple. Raises a KeyError if the dictionary is empty.

d = {100: 'Alice', 200: 'Bob', 300: 'Charlie'}
item = d.popitem()
print(item)  # Output: (300, 'Charlie') (Output may vary)
print(d)     # Remaining dictionary

3. clear() Method

Removes all items from the dictionary.

d.clear()
print(d)  # Output: {}

4. setdefault() Method

Returns the value of a key. If the key does not exist, inserts the key with the specified default value.

d = {100: 'Alice', 200: 'Bob'}
print(d.setdefault(300, 'Charlie'))  # Output: 'Charlie'
print(d)  # Output: {100: 'Alice', 200: 'Bob', 300: 'Charlie'}

# If the key exists, returns its value without changing the dictionary
print(d.setdefault(100, 'Not Alice'))  # Output: 'Alice'
print(d)  # Output: {100: 'Alice', 200: 'Bob', 300: 'Charlie'}

Aliasing and Cloning Dictionaries

Aliasing

Assigning one dictionary to another variable creates an alias. Both variables reference the same dictionary.

d1 = {100: 'Alice', 200: 'Bob'}
d2 = d1  # Aliasing
d1[300] = 'Charlie'
print(d2)  # Output: {100: 'Alice', 200: 'Bob', 300: 'Charlie'}

Cloning

Creating a copy of a dictionary that is independent of the original.

d1 = {100: 'Alice', 200: 'Bob'}
d2 = d1.copy()  # Cloning
d1[300] = 'Charlie'
print(d2)  # Output: {100: 'Alice', 200: 'Bob'}

Example: Sum of Dictionary Values

d = eval(input('Enter a dictionary: '))
total = sum(d.values())
print('The sum of values:', total)

Example: Counting Character Occurrences

Write a program to find the number of occurrences of each character in a given string.

word = input('Enter any string: ')
d = {}
for ch in word:
    d[ch] = d.get(ch, 0) + 1
print(d)

Example: Counting Vowel Occurrences

Write a program to find the number of occurrences of each vowel in a given string.

word = input('Enter any string: ').lower()
vowels = {'a', 'e', 'i', 'o', 'u'}
d = {}
for ch in word:
    if ch in vowels:
        d[ch] = d.get(ch, 0) + 1

for k, v in d.items():
    print(k, 'occurred', v, 'times')

Example: Student Marks Management

n = int(input('Enter the number of students: '))
d = {}
for i in range(n):
    name = input('Enter the student name: ')
    marks = int(input('Enter the student marks: '))
    d[name] = marks

print('Data entered successfully.')
print('*' * 30)
print('NAME', '\t\t', 'MARKS')
print('*' * 30)
for k, v in d.items():
    print(k, '\t\t', v)

print('Search operation started...')
while True:
    name = input('Enter student name to get marks: ')
    marks = d.get(name)
    if marks is None:
        print('Student not found')
    else:
        print('Marks of', name, 'is', marks)
    option = input('Do you want to find another student\'s marks? (yes/no): ')
    if option.lower() == 'no':
        break
print('Thanks for using our application')

Dictionary Comprehension

Dictionary comprehension provides a concise way to create dictionaries.

d = {x: x ** 2 for x in range(1, 6)}
print(d)  # Output: {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

Merging Collections

1. Merging Lists

Using the + operator:

l1 = [10, 20]
l2 = [30, 40]
l3 = l1 + l2
print(l3)  # Output: [10, 20, 30, 40]

Using unpacking (applicable to other collections as well):

l3 = [*l1, *l2]
print(l3)  # Output: [10, 20, 30, 40]

2. Merging Tuples

t1 = (10, 20)
t2 = (30, 40)
t3 = (*t1, *t2)
print(t3)  # Output: (10, 20, 30, 40)

3. Merging Sets

The + operator is not applicable to sets, but you can use unpacking:

s1 = {10, 20}
s2 = {30, 40}
s3 = {*s1, *s2}
print(s3)  # Output: {40, 10, 20, 30} (Order may vary)

4. Merging Dictionaries

Using the ** operator:

d1 = {100: 'A', 200: 'B'}
d2 = {300: 'C', 400: 'D'}
d3 = {**d1, **d2}
print(d3)  # Output: {100: 'A', 200: 'B', 300: 'C', 400: 'D'}

Nested Collections and Examples

You can have collections within collections, such as lists of tuples, dictionaries of lists, etc. This is especially useful in applications like web development with Django or REST APIs.

Example: List of Tuples

l = [(10, 20, 30), (40, 50, 60)]
print(l[0][1])  # Output: 20
print(l[1][1])  # Output: 50

Example: Dictionary with Tuples

d = {
    'cars': ('Innova', 'Swift', 'Brezza'),
    'mobiles': ('Samsung', 'iPhone', 'Huawei')
}

# Display the second car name
print(d['cars'][1])  # Output: 'Swift'

# Display all mobile names
for mobile in d.get('mobiles', []):
    print(mobile)

Note on Collections within Sets and Dictionaries

You cannot add mutable types like lists as elements of sets or keys of dictionaries because they are unhashable. However, you can use immutable types like tuples or use lists as values in dictionaries.
