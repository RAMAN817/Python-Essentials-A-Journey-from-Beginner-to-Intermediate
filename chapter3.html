Chapter 3: Flow Control

Introduction and Indentation

Flow control refers to the order in which individual statements, instructions, or function calls are executed or evaluated. In Python, flow control can be managed using conditional statements, loops, and other control structures.

Conditional Statements

Conditional statements allow you to execute certain code blocks based on a condition.

if Statement

Syntax:

if condition:
    statement(s)

If the condition evaluates to True, the indented statements are executed.

Example:
name = input("Enter Name: ")
if name == "durga":
    print("Hello Durga, Good Morning")
    print("How are you!!!")

Note: Indentation is crucial in Python. Failing to indent properly will result in an IndentationError.

Iterative Statements (Loops)

Iterative statements allow you to execute a block of code repeatedly.

for Loop

The for loop is used to iterate over a sequence (such as a list, tuple, string, or range).

for variable in sequence:
    statement(s)
Example:
s = input('Enter any string: ')
i = 0
for x in s:
    print("The character present at index {}: {}".format(i, x))
    i += 1
Printing a Message Multiple Times:
for x in range(10):
    print('Welcome to Python for loop')
Displaying Odd Numbers from 1 to 20:
for x in range(1, 21, 2):
    print(x)
Displaying Numbers from 10 to 1 in Descending Order:
for x in range(10, 0, -1):
    print(x)

Calculating the Sum of a List:

lst = [10, 20, 30]
print('The sum:', sum(lst))

while Loop

The while loop is used when the number of iterations is not known in advance and depends on some condition.

while condition:
    statement(s)
Example:
i = 1
while i <= 10:
    print('Hello, welcome to while loop')
    i += 1
Waiting for Specific User Input:
name = ''
while name != 'sunny':
    name = input('Enter your favourite actress: ')
print('Thanks for confirmation')
Infinite Loop:
i = 1
while True:
    print('Hello', i)
    i += 1
    # Use a break condition to exit the loop if needed

Nested Loops

A nested loop is a loop inside another loop.

Example:
for i in range(3):  # 0, 1, 2
    for j in range(2):  # 0, 1
        print('Hello')

This will print 'Hello' 6 times.

Transfer Statements

Transfer statements alter the flow of execution in a loop.

break Statement

The break statement is used to exit a loop prematurely when a certain condition is met.

Example:
for i in range(10):
    if i == 7:
        print('Processing is enough')
        break
    print(i)
Example with a Shopping Cart:
cart = [10, 20, 30, 600, 70, 80]
for item in cart:
    if item > 500:
        print("Please take insurance for item:", item)
        break
    print('Processing item:', item)

continue Statement

The continue statement skips the current iteration and moves to the next iteration of the loop.

Example:
for i in range(10):
    if i % 2 == 0:
        continue
    print(i)  # Prints odd numbers between 0 and 9
Example with a Shopping Cart:
cart = [10, 20, 30, 600, 70, 80]
for item in cart:
    if item > 500:
        print("Insurance is required, skipping item:", item)
        continue
    print('Processing item:', item)

Loop with else Block

In Python, a loop can have an else clause that executes after the loop completes normally (i.e., without encountering a break statement).

Example:
cart = [10, 20, 30, 40, 50]
for item in cart:
    if item > 500:
        print('We cannot process this order; insurance is required.')
        break
    print('Processing item:', item)
else:
    print('Congratulations, all items processed successfully!')

Special Statements

pass Statement

The pass statement is a null operation; nothing happens when it executes. It's used when a statement is syntactically required but you don't want any command or code to execute.

Example:
def some_function():
    pass  # TODO: implement this function later

del Statement

The del statement is used to delete references to objects.

Example:
x = 10
del x
# print(x)  # This would raise a NameError

Deleting a variable removes the reference to the object in memory. If other references exist, the object remains in memory.

Example with Multiple References:
s1 = 'durga'
s2 = s1
s3 = s2

del s1
print(s2)  # Output: 'durga'
print(s3)  # Output: 'durga'

In this case, the string object 'durga' is not deleted because s2 and s3 still reference it.

del vs. Immutable Objects

You cannot delete elements of immutable objects like strings.

s = 'durga'
# del s[0]  # This would raise a TypeError

del vs. None

Assigning None to a variable removes the reference to the original object but keeps the variable defined.

x = 10
x = None
print(x)  # Output: None

Prime Number Programs

Check if a Number is Prime

A prime number is a natural number greater than 1 that has no positive divisors other than 1 and itself.

Program:
n = int(input('Enter the number: '))
if n <= 1:
    print('It is not a prime number')
else:
    is_prime = True
    for i in range(2, n // 2 + 1):
        if n % i == 0:
            is_prime = False
            break
    if is_prime:
        print('It is a prime number')
    else:
        print('It is not a prime number')

Generate Prime Numbers up to a Given Number

Program:
n = int(input('Enter the value of n: '))
n1 = 2
while n1 <= n:
    is_prime = True
    for i in range(2, n1 // 2 + 1):
        if n1 % i == 0:
            is_prime = False
            break
    if is_prime:
        print(n1)
    n1 += 1

Generate First n Prime Numbers

Program:
n = int(input('Enter the number of prime numbers to generate: '))
count = 0
num = 2
while True:
    is_prime = True
    for i in range(2, num // 2 + 1):
        if num % i == 0:
            is_prime = False
            break
    if is_prime:
        print(num)
        count += 1
    if count == n:
        break
    num += 1
