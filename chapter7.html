Chapter 7: Sets

Introduction to Sets

A set in Python is an unordered collection of unique and immutable objects. Sets are mutable, meaning you can add or remove elements after creation. They are defined using curly braces {} or the set() function.

Key characteristics of sets:

  1. Duplicates are not allowed: Each element must be unique.
  2. Order is not preserved: Sets are unordered collections; the order of elements is undefined.
  3. No Indexing or Slicing: Since sets are unordered, you cannot access elements by index.
  4. Heterogeneous Objects: Elements can be of different data types.
  5. Mutable: You can add or remove elements after creation.
  6. Mathematical Set Operations: Supports union, intersection, difference, and symmetric difference.

Note: An empty pair of curly braces {} creates an empty dictionary, not a set. To create an empty set, use set().

Creating Set Objects

1. Empty Set


s = set()
print(s)        # Output: set()
print(type(s))  # Output: <class 'set'>

2. With Known Data


s = {10, 20, 30, 'hello'}
print(s)  # Output: {10, 20, 'hello', 30} (Order may vary)

3. Using the set() Function

You can create a set from any iterable object.


l = [10, 20, 30, 40, 50]
s = set(l)
print(s)  # Output: {40, 10, 50, 20, 30} (Order may vary)


s = set(range(0, 101, 10))
print(s)  # Output: {0, 40, 70, 10, 80, 50, 20, 90, 60, 100, 30} (Order may vary)


s = set('apple')
print(s)  # Output: {'a', 'p', 'l', 'e'} (Order may vary)

Operators with Sets

Mathematical Operators

Standard mathematical operators like + and * are not applicable to sets and will result in a TypeError.


s1 = {10, 20, 30}
s2 = {30, 40, 50}

# Invalid operations:
# s3 = s1 + s2  # TypeError
# s4 = s1 * 3   # TypeError

Equality Operators

Using == and != operators. Since sets are unordered, two sets are equal if they contain the same elements.


s1 = {10, 20, 30}
s2 = {30, 10, 20}

print(s1 == s2)  # Output: True
print(s1 != s2)  # Output: False

Relational Operators

Relational operators like <, >, <=, and >= are defined for sets but represent subset and superset relationships.


s1 = {10, 20}
s2 = {10, 20, 30}

print(s1 < s2)   # Output: True  (s1 is a proper subset of s2)
print(s1 > s2)   # Output: False
print(s1 <= s2)  # Output: True  (s1 is a subset of s2)
print(s1 >= s2)  # Output: False

Membership Operators

Using in and not in operators:


s = {10, 20, 30, 40, 50}
print(10 in s)        # Output: True
print(90 not in s)    # Output: True

Important Methods and Functions for Sets

1. len() Function

Returns the number of elements in the set.


s = {10, 20, 30, 40}
print(len(s))  # Output: 4

2. add() Method

Adds a single element to the set.


s = {10, 20, 30}
s.add(40)
print(s)  # Output: {40, 10, 20, 30} (Order may vary)

3. update() Method

Adds multiple elements to the set. Accepts any iterable as an argument.


s = {10, 20}
l = [30, 40, 50]
s.update(l)
print(s)  # Output: {40, 10, 50, 20, 30} (Order may vary)

You can also pass multiple iterables:


s.update(range(1, 6), 'durga')
print(s)
# Output may include numbers 1-5 and characters 'd', 'u', 'r', 'g', 'a'

4. remove() Method

Removes a specified element from the set. Raises a KeyError if the element is not found.


s = {10, 20, 30}
s.remove(20)
print(s)  # Output: {10, 30}

# s.remove(40)  # KeyError: 40

5. discard() Method

Removes a specified element from the set. Does nothing if the element is not found.


s = {10, 20, 30, 40}
s.discard(30)
print(s)  # Output: {40, 10, 20}

s.discard(50)  # No error
print(s)       # Output: {40, 10, 20}

6. pop() Method

Removes and returns an arbitrary element from the set. Raises a KeyError if the set is empty.


s = {10, 20, 30, 40}
item = s.pop()
print('Removed:', item)
print('Remaining set:', s)

Note: Since sets are unordered, you cannot predict which element will be removed.

7. clear() Method

Removes all elements from the set.


s = {10, 20, 30, 40}
s.clear()
print(s)  # Output: set()

Set Operations (Mathematical Set Operations)

1. Union

Combines elements from both sets, excluding duplicates.


s1 = {10, 20, 30, 40}
s2 = {30, 40, 50, 60}

s3 = s1.union(s2)
print(s3)  # Output: {40, 10, 50, 20, 60, 30} (Order may vary)

# Alternative syntax using '|' operator:
s3 = s1 | s2
print(s3)

2. Intersection

Includes only elements present in both sets.


s3 = s1.intersection(s2)
print(s3)  # Output: {40, 30}

# Alternative syntax using '&' operator:
s3 = s1 & s2
print(s3)

3. Difference

Elements present in the first set but not in the second set.


s3 = s1.difference(s2)
print(s3)  # Output: {10, 20}

# Alternative syntax using '-' operator:
s3 = s1 - s2
print(s3)

4. Symmetric Difference

Elements present in either set but not in both (i.e., excludes the intersection).


s3 = s1.symmetric_difference(s2)
print(s3)  # Output: {50, 20, 10, 60}

# Alternative syntax using '^' operator:
s3 = s1 ^ s2
print(s3)

Aliasing and Cloning of Sets

Aliasing

Assigning one set to another variable creates an alias; both variables reference the same set object.


s1 = {10, 20, 30, 40}
s2 = s1  # Aliasing

s1.add(50)
print(s1)  # Output: {40, 10, 50, 20, 30}
print(s2)  # Output: {40, 10, 50, 20, 30}

Cloning

Creating an exact copy of a set that is independent of the original set.


s1 = {10, 20, 30, 40}
s2 = s1.copy()  # Cloning

s1.add(50)
print(s1)  # Output: {40, 10, 50, 20, 30}
print(s2)  # Output: {40, 10, 20, 30}

Set Comprehension

Set comprehension offers a concise way to create sets.


s = {x ** 2 for x in range(1, 6)}
print(s)  # Output: {1, 4, 9, 16, 25}

Practical Examples with Sets

Example 1: Eliminating Duplicates from a List

Approach 1


l = [10, 20, 10, 20, 10, 30, 30]
s = set(l)
l_unique = list(s)
print(l_unique)  # Output: [10, 20, 30] (Order may vary)

Approach 2


l = [10, 20, 10, 20, 10, 30]
l_unique = []
for x in l:
    if x not in l_unique:
        l_unique.append(x)
print(l_unique)  # Output: [10, 20, 30]

Example 2: Finding Vowels in a Word

Write a program to print different vowels present in the given word.


word = input('Enter any word to search for vowels: ').lower()
s = set(word)
vowels = {'a', 'e', 'i', 'o', 'u'}
result = s & vowels
print('Vowels present in the word:', result)

Summary

Sets in Python are powerful data structures that allow you to store unique elements and perform efficient mathematical set operations. Remember that sets are unordered and mutable, and they cannot contain duplicate elements. Use sets when you need to eliminate duplicates or perform set-based mathematical operations.
